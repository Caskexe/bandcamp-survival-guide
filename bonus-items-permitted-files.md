#### FROM BANDCAMP HELP CENTER:

**[What file types can I include as bonus items in an album download?](https://get.bandcamp.help/hc/en-us/articles/23020664684311-What-file-types-can-I-include-as-bonus-items-in-an-album-download)**

> The usual: png, jpg, gif, pdf, doc, docx, txt, ppt, pptx, mp4, m4v,
> m4r, mov, wmv, avi, mpg, mpeg, swf, flv, torrent, cue, afm, amxd,
> maxpat, otf, sib, ptb, mid, midi, gp5, gpx, fxp, fxb, vst, mod, it,
> xm, mtm, nsf, ttf, sav, epub, mobi, vst3, and alp. Most music file
> types are deliberately not included. If you want to include a track as
> a bonus item in an album download, upload it like any other track, but
> check “bonus track.”

**[What are bonus download items and how do I use them?](https://get.bandcamp.help/hc/en-us/articles/23020709792407-What-are-bonus-download-items-and-how-do-I-use-them)**

> Bonus download items are media you can add to your digital album that
> will be included automatically in the zip file when a fan downloads
> your album. You can add PDF booklets, tabs, sheet music, videos,
> alternate art, pretty much whatever goodies you like, and we include
> them in the album’s zip file. Here are all the [file types we
> support](https://get.bandcamp.help/hc/en-us/articles/23020664684311-What-file-types-can-I-include-as-bonus-items-in-an-album-download-).
> 
> We automatically include your cover art in the zip (in the original
> size you uploaded), so no need to add that separately (unless you’re
> looking to include an even larger size).
> 
> To add a bonus download item, visit the album editor and look for the
> "add bonus item" link.
> 
> Please note: it’s not possible to add bonus download items to
> standalone tracks.

## File Types - Organised

### Documents & Text
| File Type | Description |
|----------|------------|
| pdf | Fixed-layout document |
| doc / docx | Microsoft Word documents |
| txt | Plain text file |
| ppt / pptx | PowerPoint presentations |

### Images
| File Type | Description |
|----------|------------|
| png | High-quality image with transparency |
| jpg | Compressed photo image |
| gif | Animated or simple image |

###  Video
| File Type | Description |
|----------|------------|
| mp4 | Standard modern video format |
| m4v | Apple video format |
| mov | QuickTime video |
| wmv | Windows video format |
| avi | Older video format |
| mpg / mpeg | Compressed video |
| flv | Flash video (older) |
| swf | Flash animation (deprecated) |

###  Audio & Music Files
| File Type | Description |
|----------|------------|
| mid / midi | Musical note data (not actual audio) |
| m4r | iPhone ringtone |
| mod / it / xm / mtm | Tracker music formats |
| nsf | Nintendo sound format |

###  Music Production & Plugins
| File Type | Description |
|----------|------------|
| amxd | Ableton Max plugin |
| maxpat | Max/MSP patch |
| fxp | VST preset |
| fxb | VST preset bank |
| vst / vst3 | Audio plugins |
| alp | Ableton project file |

###  Guitar & Sheet Music
| File Type | Description |
|----------|------------|
| gp5 | Guitar Pro 5 tablature |
| gpx | Guitar Pro 6+ tablature |
| ptb | Power Tab file |
| sib | Sibelius sheet music |

###  Fonts
| File Type | Description |
|----------|------------|
| otf | OpenType font |
| ttf | TrueType font |
| afm | Font metrics file |

###  Archives, Data & Misc
| File Type | Description |
|----------|------------|
| torrent | BitTorrent download file |
| cue | Disc track layout |
| sav | Game save file |

###  eBooks
| File Type | Description |
|----------|------------|
| epub | Standard eBook format |
| mobi | Kindle eBook format |
